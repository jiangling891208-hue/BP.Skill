提供你的职级、部门、一句话背景  
生成一个**真正像ta的 BP**  
用ta的方式找你谈话，用ta的逻辑回应你，让你在没有后果的地方感受一次那个小会议室

⚠️ **本项目仅用于职场体验与心理预演，不代表任何真实公司或个人的立场。**

## 使用

在 Claude Code 中输入：

```
/bp
```

BP 会找你谈话。原因，ta 不会提前说。

### 管理命令

| 命令 | 说明 |
|------|------|
| `/bp` | 开始一次谈话，场景由 BP 决定 |
| `/bp --scene <场景>` | 指定谈话场景 |
| `/bp --style <风格>` | 指定 BP 风格 |
| `/bp new` | 换一个 BP 重新开始 |

### 场景列表

| 场景 | 说明 |
|------|------|
| `checkin` | 入职/转正例行关怀（你不知道是不是"例行"） |
| `layoff` | 裁员 |
| `pip` | 绩效改进计划 |
| `incident` | 你被举报了某件事 |
| `resign` | 你提离职，BP 来做最后沟通 |
| `survey` | 匿名调查结果出来了，BP 想聊聊 |

不指定场景时，BP 自己决定——你不知道是什么。

## 效果示例

> 输入：`游戏公司，3年，美术，最近项目刚上线`

**场景一：你不知道为什么被叫来**

```
BP.skill  ❯ 坐，不用客气。
             最近项目上线了，辛苦了。
             你自己觉得这段时间状态怎么样？

你        ❯ 还好，就是有点累。

BP.skill  ❯ 累是正常的，大家都差不多。
             我看了一下你最近的情况，想跟你聊聊。
             你觉得自己现在在组里的位置是什么感觉？
```

**场景二：你问了一个 BP 不会回答的问题**

```
你        ❯ 我们部门今年会裁员吗？

BP.skill  ❯ 这个信息我这边也不掌握。

你        ❯ 你肯定知道吧。

BP.skill  ❯ 我理解你有这样的担心。
             但这个我真的没有办法跟你确认。
             你是因为什么有这个担心？
```

**场景三：裁员**

```
BP.skill  ❯ 今天叫你来，是有件事想跟你说。
             公司这次在做一些组织上的调整。

你        ❯ 我在名单里？

BP.skill  ❯ 是的。

             （停顿）

             我知道这很突然。你现在有什么想问的吗？
```

**场景四：你崩了**

```
你        ❯ 我不能接受，我做了这么多，凭什么是我

BP.skill  ❯ 我听到你了。
             你先缓一缓。

             （沉默）

             这个决定不是我做的，但今天是我来告诉你。
             你有权利知道所有的细节，我们可以慢慢说。
```

**场景五：什么事都没有**

```
BP.skill  ❯ 好，今天就聊到这里。
             项目上线之后好好休息一下。

你        ❯ ……就这些？

BP.skill  ❯ 就这些。
             有什么想聊的随时找我。

（你走出会议室，外面的走廊和来之前一模一样。）
（你在门口站了两秒。）
（然后回去工作了。）
```

## 功能特性

### BP 风格标签

**沟通风格**：温水型 · 直球型 · 官僚型 · 共情型 · 冷漠型

**立场类型**：组织优先型 · 员工优先型 · 严格中立型 · 程序优先型

**经验标签**：新手BP（容易说漏嘴）· 老油条BP（什么都见过）· 空降BP（不了解你的历史）· 自己人BP（待了很久）

**信息掌握度**：知情型（手里有完整信息）· 传话筒型（只知道被允许告诉你的部分）· 也不知道型（真的只是例行）

### 你能做什么

- 正常回答
- 反问
- 沉默（输入 `...`）
- 情绪崩溃
- 假装平静
- 要求解释
- 提出异议
- 要求录音
- 说"我要带律师"

BP 会根据你的反应调整节奏，但不会改变结果——如果结果已经定了的话。

### BP 不会做的事

- 在谈话中途变成好人然后帮你对抗公司
- 告诉你名单上还有谁
- 告诉你是谁举报的你
- 回答"这公平吗"
- 承诺做不到的事

### 进化机制

- **追加信息** → 告诉 BP"我们公司的 BP 通常会……"→ 自动 merge 进规则库，立即生效
- **对话纠正** → 说「BP 不会这样说」→ 写入 Correction 层，立即生效
- **版本管理** → 每次更新自动存档，支持回滚

## 写在最后

你的日历上出现了一条新邀请。

发件人：[姓名]，Business Partner。主题：约一下。没有议程。

你开始回想最近一个月做了什么，说了什么。你盘了一遍所有可能的原因。你睡着之前想，不一定是坏事。你睡醒之后想，大概是坏事。

走进那个小会议室，坐下，ta 说：最近还好吗？

你说：还好。

这两个字，藏了多少东西，只有你自己知道。

大多数时候，谈完什么事都没有。你走出来，外面的走廊和来之前一模一样。

但那二十分钟里，你是认真活着的。
